Quick Start.

Step 01

Install the core and bridge.

bash 
npm install react-form-engine react-form-bridge

Step 02

Initialize the hook.

tsx 
import { useReactForm } from 'react-form-bridge';

function App() {
  const { currentState, transition } = useReactForm({
    schema: mySchema,
    autoSave: true
  });
}

Step 03

Handle transitions.

tsx 
return (
  <div>
    {currentState === 'welcome' && (
      <WelcomeStep onNext={() => transition('NEXT')} />
    )}
  </div>
);

API Reference.

transition(event, data?) Move to the next state based on the schema graph. Promise<void>
back() Go back to the previous state in history. Promise<void>
updateContext(data) Merge new data into the global form context. Promise<void>
progress Calculated progress (0-100) of the current path. number
